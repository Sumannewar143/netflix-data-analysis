# Netflix Content Analysis

End-to-end data analysis of 9,957 Netflix titles using 
Python, Pandas, SQL, and Matplotlib.

## Key Findings
- Only 15% of Netflix content scores above 8/10
- 2002 content averages 7.55 rating vs 6.97 in 2019
- Documentary and Animation genres lead in quality
- Certificate data had 3,453 missing values — cleaned and handled
- Content additions peaked in 2019 with 1,288 titles

## Tools Used
Python | Pandas | SQL (SQLite) | Matplotlib | GitHub

## Files
- day1_explore.py — Dataset exploration
- day2_cleaning.py — Data cleaning and missing values
- day3_filtering.py — Filtering and sorting analysis
- day4_groupby.py — GroupBy aggregation analysis
- day5_sql.py — SQL queries using SQLite
